It is a **native macOS application**, meaning it runs smoothly on Apple Silicon and Intel Macs without requiring Java or external runtime environments.

## Key Features

### Multi-Database Support

SQLPro Studio supports a wide range of databases, including:

* MySQL and MariaDB
* PostgreSQL
* Microsoft SQL Server
* SQLite
* Oracle
* Snowflake
* Amazon Redshift

### SQL Query Editor

It provides a powerful query editor with:

* Syntax highlighting
* Auto-completion (IntelliSense-style suggestions)
* Query history
* Multiple result tabs

### Data Management Tools

Users can:

* View and edit table data directly
* Sort and filter results
* Import and export data (CSV, JSON, SQL, XML)
* Modify table structures

### Secure Connections

Supports secure database access using:

* SSH tunneling
* SSL/TLS encryption
* Stored secure credentials

### User-Friendly Interface

The interface is designed for productivity with:

* Tab-based navigation
* Database tree explorer
* Dark mode support
* Fast switching between databases

### Advanced Database Tools

Includes:

* Stored procedure execution
* User and permission management (for supported DBs)
* Database-wide search
* SQL formatting and beautification

## System Requirements

### macOS

* Intel or Apple Silicon processor
* Minimum 4 GB RAM
* Stable internet connection for remote databases
